# AI Planner Frontend

This is the frontend for AI Planner.

comming soon

## Tech stack

1. Vite: React TypeScript + SWC
2. Tailwind CSS v4
3. React Router v7
4. Zustand
5. Axios

## Current structure

```
├── src
│   ├── api/               # api requests
│   ├── assets/            # assets
│   ├── feature/           # feature components
│   ├── router/            # react-router
│   ├── shared/            # shared components
│   ├── store/             # zustand store
│   ├── App.tsx            # main app component
│   ├── global.css         # global styles
│   ├── ...
```
